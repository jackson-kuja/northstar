Northstar voice setup

Allow microphone access once.

Chrome requires the first microphone approval from a normal browser tab. Northstar will return you to your page as soon as voice access is granted.

What happens next

Use this one-time setup so the side panel and voice controls stay in the same system.

  1. Press Enable Microphone. Chrome will show its microphone prompt for this setup tab.
  2. Choose Allow. That grants Northstar access for Gemini Live voice.
  3. Return automatically. Northstar closes this tab and puts you back on your page.

If Chrome still blocks the prompt, re-enable microphone access in browser settings and run setup again.
